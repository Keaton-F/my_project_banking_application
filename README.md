# my_project_banking_application

## Описание

Простой проект для работы с банковскими картами и счетами. Программа умеет маскировать номера карты и счета, преобразовывать дату, фильтровать и сортировать операции.

## Структура

- **main.py** — примеры использования функций.
- **src/masks.py** — маскировка номеров.
- **src/processing.py** — фильтрация и сортировка.
- **src/widget.py** — преобразование даты, определение типа номера.
- **test/** — тесты (если есть).

## Как запустить

1. Открыть main.py.
2. Запустить команду:
    ```
    На данном этапе проекта, ф-ции необходимо вызывать вручную.
    ```
3. Посмотреть результат работы в консоли. 

## Пример функций

- Маскирует номер карты/счета.
- Переводит дату и время в обычный вид.
- Фильтрует список по статусу.
- Сортирует по дате.
- Фильтрует транзакции по валюте.
- Последовательно получает описания операций.
- Генерирует номера банковских карт в заданном диапазоне.

## Генераторы

- Фильтруют транзакции по валюте (filter_by_currency)
- Последовательно получают описания операций (transaction_descriptions)
- Генерируют номера банковских карт в заданном диапазоне (card_number_generator)

## Тестирование

Для запуска тестов перейдите в директорию `test` и выполните соответствующие тестовые скрипты.

Так как теперь проект покрыт тестами с помощью фреймворка **pytest**, понадобится его установка.
Чтобы установить pytest, используйте команды:

Установка через Poetry: ```poetry add --group dev pytest```

Через менеджер пакетов pip: ```python -m pip install pytest```
